# junit-poc
Demo project of usage of framework JUnit (version 4 & 5)

## Used Technologies
* Basics
  * Maven 3
  * Java 8
  * SLF4J
* Testing
  * JUnit 4.12 & JUnit 5.2.0
  * Hamcrest 1.3
  * PowerMock 2.0.0-beta.5
* Mocking
  * AssertJ 3.9.1
  * Mockito 2.18.3
  * EasyMock 3.6
* Others
  * Spring 5.0.5
  * Spring Boot 2.0.1
  * Cucumber 3.0.2

## Features
* examples for most of JUnit features
* examples of custom extensions (usage of extension model)
  * JUnit 4
    * Rule
    * Runner
    * Validation
  * JUnit 5
    * ParamResolver
    * ExtendWith
* usage of testing with Spring
  * Spring Framework
  * Spring Boot
* usage of testing with Cucumber
