# junit-poc
Demo project of usage of framework JUnit (version 4 & 5)

## Used Technologies
* Basics
  * Maven 3
  * Java 8
  * SLF4J
* Testing
  * JUnit 4.12 & JUnit 5 M4
  * Hamcrest
  * PowerMock
* Mocking
  * Mockito
  * EasyMock
* Others
  * Spring 4.3.6
  * Spring Boot 1.5.2

## Features
* examples for most of JUnit features
* examples of custom extensions (usage of extention model)
  * JUnit 4
    * Rule
    * Runner
    * Validation
  * JUnit 5
    * ParamResolver
    * ExtendWith
* usage of testing in Spring
  * Spring Framework
  * Spring Boot
